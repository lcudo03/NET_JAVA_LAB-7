# Bike Rental App – JHipster

Prosta aplikacja webowa wygenerowana w JHipsterze, służąca do zarządzania wypożyczalnią rowerów.

Projekt umożliwia:

- zarządzanie klientami,
- dodawanie profili klientów,
- obsługę rowerów i kategorii,
- tworzenie wypożyczeń,
- korzystanie z panelu administracyjnego i API.

## Technologie

- Java
- Spring Boot
- JHipster
- Angular
- Maven
- MySQL
- Docker
- Liquibase
